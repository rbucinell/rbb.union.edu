Homework Cover Sheet Attached to the Front of the Assignment

Print out the Homework Cover Sheet, fill it out completely, and attach it to the front of the homework assignment. Homeworks turned in without the Cover Sheet completely filled out and attached, will not be graded.

Homework Format

Points shown are for each problem assigned. The total number of points per problem assigned is 10

  1. Green Engineering Paper and the following header at the top of each page
    Your Name Problem #, Pg # or #
    Course Title Date
    Assignment Book Author
  2. Neatness
    1. All the text is written in capital block letters (individual gylphs).
    2. All text is neatly written and easily read.
    3. All text is properly sized.
  3. Write out the problem statement as written in the text book.
  4. GIVEN
    1. Summerize the parameters of the problem that are given in the problem statement.
    2. Only include parameters that are explicitly given in the statement.
    3. List important constraints that are stated in the problem statement.
  5. ASSUMPTIONS
    1. List all the assumptions that are relevent to the solution to the problem.
    2. List the assumptions that form the basis of the theory being applied.
  6. FIND
    1. List all the items that the problem asks you to find.
  7. FIGURE
    1. Draw the Freebody, Isometric, or Orthographic diagram that defines the problem. Use drafting tools to make the drawings.
    2. Show important geometry, loading, and constraints.
    3. Use vector notation on vector quantity.
    4. Use straight edges, compus, and templates to create an appropriate engineering diagram.
  8. SOLUTION
    1. Perform the required analysis to solve the problem.
    2. Start with Newton's and Euler's laws
    3. Perform stress analysis
    4. Make sure to briefly explain your steps as you work through the solution
    5. BOX THE FINAL ANSWER
  9. CORRECT ANSWER
    1. Correct numerical value.
    2. Answered boxed.
    3. Correct units.
    4. Correct number of significant figures.
  10. REVIEW SOLUTION
    1. Are the Units consistent and do they make sense? Are they Consistent?
    2. Does the Magnitued of the answer make sense?
    3. Is the answer Reasonable? Does it make sense?
  11. SUMMARY
    1. State the most important results using complete, but brief sentences.
    2. Comment, discuss, point out the most important assumption or constraint.
    3. State what you learned by solving this problem.
    4. Most be written using MULTIPLE, gramatically correct, sentences.
