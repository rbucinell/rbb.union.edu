Satisfactory completion of this course should demonstrate that students have:


    1. The ability to calculate and evaluate basic statistics for engineering data;
    2. The ability to design experiments that will result in data that will have statistical signifiance;
    3. The ability to apply the basic principles of statistical analysis to engineering data;
    4. The ability to evaluate the signifiance of experimental data;
    5. The ability to test engineering hypothises using statistical methods;
    6. Use modern engineering tools for statistical analysis.
